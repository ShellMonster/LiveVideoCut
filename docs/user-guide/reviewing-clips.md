# 片段审核

任务处理完成后，进入「剪辑复核」页面查看和审核所有切出的短视频片段。

![审核页面](../images/screenshots/review.png)

## 页面布局

页面以卡片网格的形式展示所有片段。每张卡片包含以下信息：

- **缩略图**：片段封面图
- **时长**：片段的播放时长
- **商品名称**：AI 识别的商品名
- **审核状态**：当前片段的审核进度

页面顶部有筛选标签，可以按状态快速过滤：

| 标签 | 含义 |
|------|------|
| 全部 | 显示所有片段 |
| 待审核 | 尚未处理的片段 |
| 已通过 | 确认可用的片段 |
| 需调整 | 需要重新导出的片段 |
| 已跳过 | 不需要的片段 |

## 审核操作

点击某张卡片后，右侧会弹出详情抽屉，你可以在里面执行四种操作：

| 操作 | 说明 |
|------|------|
| ✅ 通过 | 确认这个片段可以直接使用 |
| ⏭ 跳过 | 这个片段不需要，跳过 |
| ⚠️ 需调整 | 片段有问题，需要修改后重新导出 |
| 🔄 重导出 | 保留原始切割，只重新处理字幕和导出 |

## 详情抽屉

右侧抽屉包含三个页签：

### 详情

展示片段的商品信息、时长指标，以及快捷操作按钮。

### 字幕

可以逐行编辑字幕文本和起止时间。每行字幕可以修改：

- **字幕文本**：直接编辑文字内容
- **开始时间**：该行字幕出现的秒数
- **结束时间**：该行字幕消失的秒数

修改完成后点击保存，然后点击「重导出」即可生成带新字幕的视频。

> 字幕编辑不会修改原始转写结果，所有覆盖内容单独保存，可以随时恢复。

### 建议

AI 提供的优化建议，比如商品描述优化、片段时长调整等参考信息。

## 单片段重导出

当你修改了字幕后，可以对单个片段执行重导出：

1. 在字幕页签完成编辑，点击保存
2. 点击「重导出」按钮
3. 系统会用新的字幕内容重新烧录并导出视频
4. 导出完成后页面会自动更新

## 批量下载

已通过的片段支持批量下载为 ZIP 压缩包：

1. 选择需要下载的片段（仅已通过状态可下载）
2. 点击底部操作栏的「下载」按钮
3. 系统打包后自动下载

> 单次批量下载最多支持 20 个片段。超过 20 个时分批操作。
