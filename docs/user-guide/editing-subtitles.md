# 编辑字幕

视频处理完成后，你可能会发现某些片段的字幕不够准确。也许是转写错了几个字，也许是时间点偏了半秒。没关系，系统提供了字幕编辑功能，你可以修改字幕内容并重新导出。

## 核心原则：原始数据永远不改

这是最重要的一点：**你修改的字幕是"草稿"，原始的转写结果始终保留**。

打个比方，原始转写结果就像一份合同原件，你改的是复印件上的批注。原件不会动，你的批注随时可以撤回重来。

这意味着：

- 你可以放心大胆地改，不用担心改乱了回不去
- 每个片段的原始字幕和你的修改是分开保存的
- 如果你想放弃修改，直接删除草稿就能恢复原样

## 编辑步骤

1. 进入左侧导航栏的"剪辑复核"页面

   ![剪辑复核页面](../images/screenshots/review.png)

2. 找到你想编辑的片段，点击它打开右侧的复核抽屉
3. 在抽屉顶部切换到"字幕"页签
4. 你会看到这个片段的所有字幕行，每行包含：
   - **文本**：字幕显示的文字
   - **开始时间**：这行字幕出现的时刻（单位：秒）
   - **结束时间**：这行字幕消失的时刻（单位：秒）
5. 直接点击文字就能修改内容，点击时间就能调整起止点
6. 改完后点击"保存草稿"

> **保存草稿不会立即生成新视频**。它只是记下了你的修改。你需要手动触发重导出。

## 重新导出

保存草稿后，点击"重导出"按钮。系统会用你修改后的字幕重新生成这个片段的视频。

重导出只处理单个片段，不会影响其他片段，也不需要重新跑整个处理流程。通常几十秒到一两分钟就能完成。

## 时间格式说明

字幕的时间是**秒数**，不是常见的"分:秒"格式。

- 5 秒 → 写 `5.00`
- 1 分 30 秒 → 写 `90.00`
- 3 分 05 秒 25 → 写 `185.25`

保留 2 位小数就够了。系统会自动帮你处理多余的位数。

### 时间校验规则

保存时系统会检查以下几点：

- 结束时间必须大于开始时间（不能倒着来）
- 文本为空的行会被自动过滤掉
- 单行文本不能太长（建议不超过 200 字）

如果某一行的时间校验不通过，系统会提示你具体是哪一行有问题，改好再保存就行。

## 批量编辑工作流

如果你有多个片段需要修改字幕，推荐这样做：

1. 逐个打开片段的复核抽屉
2. 切到"字幕"页签，编辑文字和时间
3. 每个片段改完后点"保存草稿"（只保存，不急着导出）
4. 把所有需要改的片段都改完
5. 回过头逐个点"重导出"

这样做的好处是，你可以一次性把所有编辑工作做完，心里有数，再统一导出。避免改一个导一个，来回等。

## 常见问题

**改完字幕后，其他片段会受影响吗？**

不会。每个片段的字幕修改是独立的，只影响当前片段。

**重导出后还能继续改字幕吗？**

可以。改完再保存草稿、再重导出，没有次数限制。

**不小心改错了怎么办？**

清空你的草稿内容，恢复成和原始字幕一样的文字，保存草稿后重导出，就能得到和最初一样的结果。

**卡拉 OK 字幕也可以编辑吗？**

可以。编辑的是字幕的文本内容和起止时间，卡拉 OK 的逐字高亮效果会自动根据新的文本重新生成。

**时间点怎么找？**

一个简单的方法：在预览区播放视频，看到字幕出现或消失的时候记下进度条上的时间，填到对应的位置就行。
