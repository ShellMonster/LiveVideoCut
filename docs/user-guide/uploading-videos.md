# 上传视频

## 视频要求

上传前，请确认你的视频文件满足以下条件：

| 要求     | 说明                                   |
| -------- | -------------------------------------- |
| 格式     | MP4                                    |
| 视频编码 | H.264                                  |
| 音频编码 | AAC                                    |
| 音频流   | 必须包含（纯无声视频无法转写字幕）     |
| 文件大小 | 建议不超过 2GB，过大的文件上传时间较长 |

如果你的视频不是 MP4 格式，可以先用格式转换工具（如剪映、格式工厂）转成 MP4 再上传。

## 四种预设方案

上传时会让你选择一种预设方案。不同预设的核心区别在于"花多少钱、出什么效果"，选对了能省钱又省时间。

| 预设           | 语音转写 | 字幕样式 | AI 智能确认 | 背景音乐 | 适用场景                     |
| -------------- | -------- | -------- | ----------- | -------- | ---------------------------- |
| 高质量字幕版   | 火山 VC  | 卡拉 OK  | 开启        | 开启     | 最终成品，可以直接发布       |
| 快速低成本版   | 阿里云   | 基础     | 关闭        | 关闭     | 快速预览切分效果，测试用     |
| 全量候选调试版 | 阿里云   | 基础     | 关闭        | 关闭     | 调试切分参数，看全部候选片段 |
| 纯切片无字幕版 | 关闭     | 无       | 关闭        | 关闭     | 只需要切片，不需要字幕和转写 |

### 怎么选？

- **第一次用**：选"快速低成本版"，先看看切分效果满不满意。
- **准备发布**：选"高质量字幕版"，卡拉 OK 逐字高亮字幕效果好，可以直接发抖音、小红书。
- **调参数**：选"全量候选调试版"，会导出所有候选片段，方便你判断哪里切多了或切少了。
- **只要切片**：选"纯切片无字幕版"，跳过转写和字幕，最快最便宜。

> **关于费用**：语音转写是唯一产生外部 API 费用的环节。"火山 VC"效果最好但最贵（约 6.5 元/小时），"阿里云"效果一般但便宜（约 0.29 元/小时）。纯切片模式不调用任何外部服务，零费用。

## 上传步骤

1. 打开左侧导航栏，点击"创建任务"
2. 把视频文件拖进上传区域，或者点击区域选择文件

   ![创建任务页面](../images/screenshots/create.png)

3. 选择一种预设方案
4. 点击"开始上传"

上传完成后，页面会自动跳转到任务队列，你可以实时看到处理进度。

## 上传后会发生什么

视频上传后系统会自动开始处理，不需要你做任何操作。整个处理流程大致是：

1. **抽帧分析**：从视频中每隔一段时间抽取画面，检测主播换衣服的节点
2. **AI 确认**（如果开启了）：用 AI 判断每个换衣节点是不是真的换了商品
3. **语音转写**：把视频中的语音转成文字，带时间戳
4. **导出片段**：按照切分结果，把每个商品讲解片段单独切出来，烧上字幕

整个过程根据视频长度和所选预设，通常需要几分钟到十几分钟不等。你可以在任务队列页面实时查看进度。

## 常见问题

**上传失败怎么办？**

检查视频格式是否是 MP4，是否有声音。如果文件太大导致上传超时，可以尝试压缩后再上传。

**可以同时上传多个视频吗？**

目前一次只能上传一个视频。如果有多个视频，请依次上传，每个视频会生成一个独立的任务。

**上传后发现预设选错了怎么办？**

很遗憾，预设方案在上传时就确定了，无法修改。你需要等当前任务处理完后，用正确的预设重新上传一次。
