# 费用估算

处理一段直播视频需要调用多种 AI 服务。本文档帮你估算不同配置下的成本。

> 以下费用基于 20 分钟的直播视频估算，实际成本因视频长度和切出片段数量而异。

## 语音转写（ASR）

语音转写是必选项，根据你选择的转写服务计费：

| 服务商 | 计费标准 | 20分钟视频成本 |
|--------|----------|---------------|
| DashScope（阿里云） | ¥0.29/小时 | ≈¥0.10 |
| 火山引擎 大模型标准版 | ¥2.3/小时 | ≈¥0.77 |
| 火山引擎 VC 字幕版 | ¥6.5/小时 | ≈¥2.17 |

VC 字幕版使用剪映引擎分句，效果最好，推荐对字幕质量有要求的场景使用。

## 视觉识别（VLM）

VLM 用于识别每个候选片段的商品内容。仅在「智能模式」下启用。

- 依赖服务商：通义千问（Qwen）或智谱（GLM）
- 每个候选片段调用一次，20 分钟视频通常有 10-15 个候选
- 单次视频 VLM 费用约 **¥0.5 ~ ¥2**

如果你选择「无 VLM 模式」或「全部导出模式」，这部分费用为零。

## LLM 文本分析

LLM 用于分析转写文本，识别换品边界和优化片段切割。默认关闭，需要手动开启。

- 单次视频约 **¥0.3 ~ ¥1**
- 包含文本边界分析和边界精修两次调用

## AI 商品素材

从片段封面图生成商品营销素材，包括商品识别、文案撰写和模特图生成。

- **Gemini 商品识别 + 文案**：约 ¥0.5 ~ ¥1 / 片段
- **OpenAI 图片生成**：约 ¥1.5 ~ ¥4 / 片段（生成 4 张图）
- 合计每个片段约 **¥2 ~ ¥5**

这个功能是可选的，只有在片段资产页手动触发才会产生费用。

## 三种成本方案

### 方案一：最低成本

适合只需要快速切片段、不烧字幕的场景。

| 配置项 | 选择 |
|--------|------|
| 转写服务 | DashScope |
| 导出模式 | 无 VLM 模式 |
| 字幕 | 关闭 |
| LLM | 关闭 |
| **单视频成本** | **≈¥0.1** |

### 方案二：推荐配置

适合日常使用，效果和成本平衡。

| 配置项 | 选择 |
|--------|------|
| 转写服务 | 火山引擎 VC 字幕版 |
| 导出模式 | 智能模式 |
| 字幕 | 卡拉OK 逐字高亮 |
| 背景音乐 | 开启 |
| **单视频成本** | **≈¥3 ~ ¥5** |

### 方案三：全功能

适合需要完整商品素材输出的场景。

在推荐配置基础上增加：

| 额外功能 | 费用 |
|----------|------|
| LLM 文本分析 | +¥0.5 ~ ¥1 |
| AI 商品素材（5 个片段） | +¥10 ~ ¥25 |
| **单视频总成本** | **≈¥8 ~ ¥15** |

## 省钱建议

1. **先用无 VLM 模式试水**，确认切割效果满意后再开启智能模式
2. **DashScope 转写免费额度较高**，测试阶段可以用它
3. **AI 商品素材按需触发**，不需要的片段不要生成
4. **关闭 LLM 文本分析**，如果纯视觉切割效果已经够好
